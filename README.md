# Capstone_2024
Personal Data Sharing Capstone by Marco Li, Tish Zhou, and Evelyn Feng

This repository contains code for RL simulation used for A Game-Theoretic Formulation of Personal Data Sharing. We formulated multiple users' data sharing in a recommendation system as an incomplete-information game with multiple agents and ran simulations with different versions of DQNs to study the Nash Equilibrium and optimal strategies to balance personalization and privacy.
